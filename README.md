# RISC_V_Superscalar_Luis_Mena
RISC_V_Superscalar_Luis_Mena

Las instrucciones se cargan a la cache por medio de un archivo txt
El formato de la cache es el siguiente:
I4 I3 I2 I1
I8 I7 I6 I6

Ejemplo (la instrucción 1 siendo la de la derecha y la 4 la de la izquierda): 
00200113 00100093 00000013 00000013

En la carpeta de questa_sim se encuentra el archivo wave.do para agregar las señales a la simulación.
